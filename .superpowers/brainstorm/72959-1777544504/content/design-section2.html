Design — Section 2 of 3: Player Profile Modal

Click a bounding box on the video → modal appears with full player stats

2A — Click Detection on Canvas

Problem today

Canvas has pointer-events:none — clicks pass through to the video element (play/pause). No click handler exists.

Fix

Canvas gets pointer-events:auto + onClick handler. On click: compute canvas-relative coords → check if point is inside any player bounding box → if yes, open modal for that player. If no box hit → pass click to video (toggle play).

2B — New API Endpoint

GET /api/matches/{match_id}/players/{player_id}/stats

Returns:
 • presence: frames_detected, total_frames, involvement_pct
 • actions: per action_type → { total, success, error, neutral }
 • zones: count per zone 1–6 (from actions.zone)
 • efficiency: attack_eff, serve_eff
 • timeline: list of { timestamp, action_type, result } — last 10 actions
